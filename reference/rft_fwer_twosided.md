# Two-sided RFT wrapper

Two-sided RFT wrapper

## Usage

``` r
rft_fwer_twosided(stat_vol, method = c("peak", "cluster"), ...)
```

## Arguments

- stat_vol:

  NeuroVol containing statistic map

- method:

  "peak" or "cluster"

- ...:

  Additional arguments passed to rft_peak_fwer or rft_cluster_fwer
